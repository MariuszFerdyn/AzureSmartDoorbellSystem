## Install Software

Copy the repository to your Raspberry:
```
cd /opt
sudo git clone https://github.com/MariuszFerdyn/AzureSmartDoorbellSystem.git
sudo chown -R $USER:$USER /opt/AzureSmartDoorbellSystem
```
